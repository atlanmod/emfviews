Installation

To install/use EMF-Views, we recommend using:

  1. The modeling distribution of Eclipse (Kepler), that can be found here.
  2. The Epsilon EMF framework and Epsilon Correspondence Language, that can be found in this update site.

Once your Eclipse is ready, you can proceed with the installation of EMF-Views by:

  1. Clicking on Help / Install New Software... here.
  2. Adding a new Software Site. Click on Add... and fill in the form as indicated:(the update site is http://jdvillacalle.com/emfviews/update/site.xml).
  3. You should now see the EMF-Views plugin in the list of components to be installed. Select it and click on Next:
  4. You can now quickly accept the rest of the steps (license, etc...) and reboot Eclipse
